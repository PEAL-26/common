# Common Packages (monorepo)

Biblioteca de utilitários, componentes, hooks e helpers JavaScript/TypeScript pensada para uso em aplicações web e Node, com foco especial em padrões e necessidades de Angola.

## Visão geral

Este repositório contém um conjunto de pacotes reutilizáveis:

- Components (UI)
- Helpers (ex.: formatação de moeda)
- Hooks React (ex.: detecção de clique fora de um elemento)
- Scripts e configurações deste monorepo

### Instalação

Instale o package principal ou importe os pacotes individuais a partir do seu registry interno ou via link local:

```sh
# exemplo genérico
npm install @seu-scope/common
```
